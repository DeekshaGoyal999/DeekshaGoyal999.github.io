Assignment 1

Nov 29, 2018

1. Draw a figure as given in the file “cartoon.png”. Save the file as “object1.jpg”.
2. Draw the scene as given in the file “scene1.jpg”. Save the file as “object2.jpg”.
3. Create a complete picture by combining “object1.jpg” and “object2.jpg”. Save the file as “scene2.jpg”.
4. Open the file “cartoon.png”.
a. Rotate the figure to 180 degree.
b. Rotate the figure to 90 degree.
Save the file as “rotate1.jpg”.
5. Again open the file “cartoon.png”
a. Scale the figure to 50%.
b. Scale the figure to 150%.
Save the file as “scale.jpg”.
6. Open the file “cartoon.png”. Create a presentation where the figure is moving from lower left corner to upper right corner. The figure must rotate and increase in size while moving.
a. Save the file as “scene3.fla”.
b. Export the file as “scene3.swf”.

    • Assignment 2

    1. Draw a clock. Rotate the hour’s hand and minute’s hand. Save the work as “clock_rotate.fla”. And export your work as “clock_rotate.swf”.
    2. Draw a circle with fill color red. Move the circle on the stage from the top left corner to bottom right corner then to the center of the stage. The total animation will be of 10 seconds. Save the file as “motion.fla”. Play the movie.
    3. Animate the shape of the circle morphing to a rectangle and then to a triangle after every 4 seconds. Save the file as “morph.fla”.
    4. Create a circle. After two seconds the circle morphing to rectangle. Again after two seconds the rectangle morphing to triangle. Again after two seconds the triangle morphing to square. Again after two seconds the square morphing to ellipse. The circle fill color will be red, the rectangle fill color will be yellow, the triangle fill color will be green, the square fill color will be blue and the ellipse fill color will be brown. Save the work as “shape_color_tween.fla”. Export your work as “shape_color_tween.swf”.
